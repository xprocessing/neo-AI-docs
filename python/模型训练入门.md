### 二、示例1：传统机器学习模型（逻辑回归）
鸢尾花数据集是经典的分类数据集，包含3类鸢尾花（标签0/1/2），每朵花有4个特征（花萼长度、花萼宽度、花瓣长度、花瓣宽度）。

```python
# 1. 导入必要的库
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LogisticRegression
from sklearn.metrics import accuracy_score

# 2. 加载数据集
iris = load_iris()
X = iris.data  # 特征数据（4个特征）
y = iris.target  # 标签（3类鸢尾花：0/1/2）

# 3. 划分训练集和测试集（70%训练，30%测试）
X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.3, random_state=42  # random_state保证结果可复现
)

# 4. 初始化模型（逻辑回归）
model = LogisticRegression(max_iter=200)  # max_iter增大避免收敛问题

# 5. 训练模型
model.fit(X_train, y_train)

# 6. 模型评估（计算测试集准确率）
y_pred = model.predict(X_test)
accuracy = accuracy_score(y_test, y_pred)
print(f"逻辑回归模型测试集准确率：{accuracy:.2f}")

# 7. 示例预测
sample = [[5.1, 3.5, 1.4, 0.2]]  # 第一类鸢尾花的特征
pred = model.predict(sample)
print(f"示例样本预测类别：{iris.target_names[pred[0]]}")
```


### 三、示例2：简单神经网络模型（Keras）
用TensorFlow的Keras构建一个浅层神经网络，同样解决鸢尾花分类问题：

```python
# 1. 导入必要的库
import tensorflow as tf
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Dense
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split

# 2. 加载并划分数据集
iris = load_iris()
X = iris.data
y = iris.target
X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.3, random_state=42
)

# 3. 构建神经网络模型
model = Sequential([
    Dense(10, activation='relu', input_shape=(4,)),  # 隐藏层：10个神经元，ReLU激活
    Dense(3, activation='softmax')  # 输出层：3个神经元（对应3类），Softmax激活
])

# 4. 编译模型（设置优化器、损失函数、评估指标）
model.compile(
    optimizer='adam',  # 常用优化器
    loss='sparse_categorical_crossentropy',  # 标签为整数时用此损失
    metrics=['accuracy']
)

# 5. 训练模型
history = model.fit(
    X_train, y_train,
    epochs=50,  # 训练轮数
    batch_size=5,  # 每次更新参数的样本数
    validation_split=0.1  # 用10%的训练数据做验证
)

# 6. 模型评估
test_loss, test_acc = model.evaluate(X_test, y_test)
print(f"神经网络模型测试集准确率：{test_acc:.2f}")

# 7. 示例预测
sample = [[6.3, 2.9, 5.6, 1.8]]  # 第三类鸢尾花的特征
pred_probs = model.predict(sample)
pred_class = tf.argmax(pred_probs, axis=1).numpy()[0]
print(f"示例样本预测类别：{iris.target_names[pred_class]}")
```


### 核心流程总结
无论是传统机器学习还是神经网络，模型训练的核心步骤都是：  
**数据加载与预处理 → 划分训练/测试集 → 模型定义 → 模型训练 → 模型评估 → 预测**  

以上示例足够简单且可复现，你可以在此基础上尝试调整模型参数（如神经网络的层数、神经元数量）或换用其他数据集（如`sklearn`的`load_wine`、`load_digits`）来加深理解。
