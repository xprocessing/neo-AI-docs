你希望我提供一段高度浓缩、带详细注解的HTML核心标签代码，内容精炼到能在一张A4纸上打印完成，方便学习和查阅。

下面是为你整理的HTML核心标签汇总代码，涵盖新手最常用的标签类型，注解详尽且排版紧凑，完全可以在A4纸上清晰打印：

```html
<!DOCTYPE html>
<!-- 声明文档类型为HTML5 -->
<html lang="zh-CN">
<!-- 根元素，lang指定页面语言为中文 -->
<head>
    <!-- 头部：页面元信息，不显示在页面主体 -->
    <meta charset="UTF-8">
    <!-- 字符编码：UTF-8兼容所有字符 -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <!-- 视口设置：适配移动端 -->
    <title>HTML核心标签速查</title>
    <!-- 页面标题：显示在浏览器标签栏 -->
    <style>
        /* 简单样式：优化打印效果，可根据需要调整 */
        body { font-family: Consolas, "微软雅黑"; font-size: 12px; line-height: 1.4; }
        .container { max-width: 800px; margin: 0 auto; padding: 10px; }
        h1, h2 { margin: 8px 0; }
        .tag-demo { margin: 5px 0; padding: 5px; border-left: 3px solid #ccc; }
        code { background: #f5f5f5; padding: 2px 4px; }
    </style>
</head>
<body>
    <div class="container">
        <!-- 1. 基础结构标签 -->
        <h2>1. 基础结构标签</h2>
        <div class="tag-demo">
            <code>&lt;html&gt;</code> - 文档根元素<br>
            <code>&lt;head&gt;</code> - 页面元信息容器<br>
            <code>&lt;body&gt;</code> - 页面可见内容容器<br>
            <code>&lt;title&gt;</code> - 页面标题（浏览器标签）<br>
            <code>&lt;meta&gt;</code> - 元数据（编码、视口、关键词等）
        </div>

        <!-- 2. 文本排版标签 -->
        <h2>2. 文本排版标签</h2>
        <div class="tag-demo">
            <h1>一级标题 &lt;h1&gt;</h1>
            <h2>二级标题 &lt;h2&gt;</h2>
            <h6>六级标题 &lt;h6&gt;</h6>
            <p>段落标签 &lt;p&gt;：这是一个普通段落，用于包裹一段文本内容。</p>
            <br> - 换行标签 &lt;br&gt;<br>
            <hr> - 水平线标签 &lt;hr&gt;<br>
            <strong>加粗文本 &lt;strong&gt;</strong><br>
            <em>斜体文本 &lt;em&gt;</em><br>
            <span>行内容器 &lt;span&gt;：用于包裹行内小范围文本</span><br>
            <div>块级容器 &lt;div&gt;：用于划分页面区域（独占一行）</div>
        </div>

        <!-- 3. 列表标签 -->
        <h2>3. 列表标签</h2>
        <div class="tag-demo">
            <!-- 无序列表 -->
            <h4>无序列表 &lt;ul&gt; + &lt;li&gt;</h4>
            <ul>
                <li>列表项1</li>
                <li>列表项2</li>
            </ul>

            <!-- 有序列表 -->
            <h4>有序列表 &lt;ol&gt; + &lt;li&gt;</h4>
            <ol>
                <li>第一步</li>
                <li>第二步</li>
            </ol>

            <!-- 定义列表 -->
            <h4>定义列表 &lt;dl&gt; + &lt;dt&gt; + &lt;dd&gt;</h4>
            <dl>
                <dt>HTML</dt>
                <dd>超文本标记语言</dd>
                <dt>CSS</dt>
                <dd>层叠样式表</dd>
            </dl>
        </div>

        <!-- 4. 链接与多媒体标签 -->
        <h2>4. 链接与多媒体标签</h2>
        <div class="tag-demo">
            <a href="https://www.baidu.com" target="_blank">链接标签 &lt;a&gt;：百度（新窗口打开）</a><br>
            <!-- href: 链接地址, target="_blank": 新窗口打开 -->
            
            <img src="https://picsum.photos/100/80" alt="示例图片" title="图片标题">
            <!-- 图片标签 &lt;img&gt;：src=图片地址, alt=替代文本(加载失败/无障碍), title=鼠标悬浮提示 -->
            
            <audio controls>
                <!-- 音频标签 &lt;audio&gt;：controls显示控制栏 -->
                <source src="audio.mp3" type="audio/mpeg">
                您的浏览器不支持音频播放
            </audio>

            <video controls width="200">
                <!-- 视频标签 &lt;video&gt;：controls显示控制栏, width设置宽度 -->
                <source src="video.mp4" type="video/mp4">
                您的浏览器不支持视频播放
            </video>
        </div>

        <!-- 5. 表格标签 -->
        <h2>5. 表格标签</h2>
        <div class="tag-demo">
            <table border="1" cellpadding="5" cellspacing="0">
                <!-- border: 边框, cellpadding: 内边距, cellspacing: 单元格间距 -->
                <thead>
                    <tr> <!-- 表格行 &lt;tr&gt; -->
                        <th>表头1 &lt;th&gt;</th> <!-- 表头单元格 -->
                        <th>表头2</th>
                    </tr>
                </thead>
                <tbody>
                    <tr>
                        <td>单元格1 &lt;td&gt;</td> <!-- 普通单元格 -->
                        <td>单元格2</td>
                    </tr>
                </tbody>
            </table>
        </div>

        <!-- 6. 表单标签 -->
        <h2>6. 表单标签</h2>
        <div class="tag-demo">
            <form action="/submit" method="post">
                <!-- action: 提交地址, method: 提交方式(get/post) -->
                
                <label for="username">用户名：</label>
                <input type="text" id="username" name="username" placeholder="请输入用户名" required>
                <!-- type: 输入类型, id/for: 关联标签, name: 提交参数名, placeholder: 提示文字, required: 必填 -->
                <br>

                <label for="pwd">密码：</label>
                <input type="password" id="pwd" name="pwd">
                <!-- password: 密码输入框（隐藏输入内容） -->
                <br>

                <label>性别：</label>
                <input type="radio" name="gender" value="male" checked> 男
                <input type="radio" name="gender" value="female"> 女
                <!-- radio: 单选框, name相同为一组, checked: 默认选中 -->
                <br>

                <label>爱好：</label>
                <input type="checkbox" name="hobby" value="read"> 阅读
                <input type="checkbox" name="hobby" value="sport"> 运动
                <!-- checkbox: 复选框 -->
                <br>

                <label for="city">城市：</label>
                <select id="city" name="city">
                    <!-- 下拉选择框 -->
                    <option value="">请选择</option>
                    <option value="beijing" selected>北京</option>
                    <option value="shanghai">上海</option>
                </select>
                <br>

                <label for="desc">描述：</label>
                <textarea id="desc" name="desc" rows="2" cols="30"></textarea>
                <!-- 多行文本框, rows: 行数, cols: 列数 -->
                <br>

                <input type="submit" value="提交">
                <input type="reset" value="重置">
                <!-- submit: 提交按钮, reset: 重置按钮 -->
            </form>
        </div>

        <!-- 7. 语义化标签 (HTML5) -->
        <h2>7. HTML5语义化标签</h2>
        <div class="tag-demo">
            <header>页面头部 &lt;header&gt;</header>
            <nav>导航栏 &lt;nav&gt;：包含页面导航链接</nav>
            <main>主要内容 &lt;main&gt;：页面核心内容（唯一）</main>
            <section>章节 &lt;section&gt;：文档中的一个章节</section>
            <article>文章 &lt;article&gt;：独立的内容块（如文章、评论）</article>
            <aside>侧边栏 &lt;aside&gt;：辅助内容（如广告、推荐）</aside>
            <footer>页面底部 &lt;footer&gt;</footer>
        </div>
    </div>
</body>
</html>
```

### 打印建议
1. 将代码复制到VS Code/记事本，设置等宽字体（如Consolas），字号10-11号
2. 打印时选择“仅打印HTML”（或直接在浏览器中打开该文件后打印），取消页眉页脚
3. 调整页边距为1cm，选择“适应宽度”打印，可隐藏浏览器的打印背景（减少墨量）
