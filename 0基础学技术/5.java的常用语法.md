### 一、基础语法：变量、数据类型与运算符
这是Java程序的最小组成单元，也是最基础的部分。

#### 1. 数据类型
Java是强类型语言，变量必须先声明类型再使用，分为两大类：
- **基本数据类型**（8种）：`byte`、`short`、`int`、`long`（整数）；`float`、`double`（浮点）；`char`（字符）；`boolean`（布尔）。
- **引用数据类型**：字符串（`String`）、数组、类、接口等。

#### 2. 变量声明与赋值
```java
public class BasicSyntax {
    public static void main(String[] args) {
        // 1. 基本数据类型变量
        int age = 20; // 整数，最常用
        double score = 95.5; // 浮点型，存小数
        char gender = '男'; // 字符，单引号，只能存一个字符
        boolean isStudent = true; // 布尔型，只有true/false

        // 2. 引用数据类型（字符串）
        String name = "张三"; // 字符串，双引号

        // 3. 输出变量
        System.out.println("姓名：" + name);
        System.out.println("年龄：" + age);
        System.out.println("成绩：" + score);
    }
}
```

#### 3. 常用运算符
包括算术运算符（`+`、`-`、`*`、`/`、`%`）、赋值运算符（`=`、`+=`）、比较运算符（`==`、`!=`、`>`、`<`）、逻辑运算符（`&&`、`||`、`!`）。
```java
public class Operators {
    public static void main(String[] args) {
        int a = 10, b = 3;
        
        // 算术运算
        System.out.println(a + b); // 13
        System.out.println(a % b); // 1（取余）
        
        // 比较运算 + 逻辑运算
        boolean result = (a > 5) && (b < 5); // true
        System.out.println(result);
    }
}
```

### 二、流程控制：条件、循环
控制程序的执行顺序，是实现逻辑的核心。

#### 1. 条件语句（if-else / switch）
```java
public class Condition {
    public static void main(String[] args) {
        int score = 85;
        
        // if-else
        if (score >= 90) {
            System.out.println("优秀");
        } else if (score >= 80) {
            System.out.println("良好");
        } else {
            System.out.println("及格");
        }
        
        // switch（JDK 12+支持表达式写法）
        int week = 1;
        switch (week) {
            case 1:
                System.out.println("周一");
                break; // 跳出switch，否则会穿透
            case 2:
                System.out.println("周二");
                break;
            default:
                System.out.println("其他");
        }
    }
}
```

#### 2. 循环语句（for / while / do-while）
```java
public class Loop {
    public static void main(String[] args) {
        // 1. for循环（最常用，适合已知循环次数）
        for (int i = 1; i <= 5; i++) {
            System.out.println("循环次数：" + i);
        }
        
        // 2. while循环（适合未知循环次数）
        int j = 1;
        while (j <= 3) {
            System.out.println("while循环：" + j);
            j++;
        }
        
        // 3. do-while（至少执行一次）
        int k = 1;
        do {
            System.out.println("do-while循环：" + k);
            k++;
        } while (k <= 2);
    }
}
```

### 三、数组：存储多个同类型数据
```java
public class ArrayDemo {
    public static void main(String[] args) {
        // 1. 数组声明与初始化
        int[] nums = {1, 2, 3, 4, 5}; // 静态初始化
        String[] names = new String[3]; // 动态初始化（指定长度）
        names[0] = "张三";
        names[1] = "李四";
        
        // 2. 遍历数组（普通for + 增强for）
        // 普通for（可操作索引）
        for (int i = 0; i < nums.length; i++) {
            System.out.println(nums[i]);
        }
        
        // 增强for（仅遍历，不能修改元素）
        for (String name : names) {
            System.out.println(name);
        }
    }
}
```

### 四、面向对象核心语法（Java的核心）
Java是纯面向对象语言，类和对象是核心。

#### 1. 类与对象
```java
// 定义类（Person）
class Person {
    // 成员变量（属性）
    String name;
    int age;
    
    // 成员方法（行为）
    public void sayHello() {
        System.out.println("我是" + name + "，今年" + age + "岁");
    }
}

// 测试类
public class OopDemo {
    public static void main(String[] args) {
        // 创建对象（实例化）
        Person p1 = new Person();
        // 给属性赋值
        p1.name = "张三";
        p1.age = 20;
        // 调用方法
        p1.sayHello(); // 输出：我是张三，今年20岁
    }
}
```

#### 2. 方法重载（同一类中，方法名相同、参数列表不同）
```java
class Calculator {
    // 加法：两个int
    public int add(int a, int b) {
        return a + b;
    }
    
    // 加法：三个int（重载）
    public int add(int a, int b, int c) {
        return a + b + c;
    }
    
    // 加法：两个double（重载）
    public double add(double a, double b) {
        return a + b;
    }
}
```

### 总结
1. Java核心基础语法包含：**变量/数据类型**（强类型）、**流程控制**（if/for/while）、**数组**（存储同类型数据）、**面向对象**（类/对象/方法重载）四大核心模块。
2. 语法规则上要注意：Java区分大小写、语句以分号结尾、花括号`{}`包裹代码块、数组索引从0开始。
3. 面向对象是Java的核心特征，类是模板，对象是类的实例，方法是类的行为。

这些是Java最常用、最基础的语法，掌握后就能编写简单的Java程序，后续可进一步学习继承、多态、异常处理、集合等进阶内容。
