# Java Spring Boot 从入门到精通学习路线

## 基础准备
- Java 基础
  - 面向对象（封装、继承、多态）
  - 集合框架（List、Map、Set）
  - 异常处理
  - 泛型 & 注解
  - Lambda 表达式 & Stream API
- 开发环境搭建
  - JDK 安装与配置（建议 JDK 17+）
  - IDE：IntelliJ IDEA / VS Code
  - Maven / Gradle 构建工具
- 数据库基础
  - SQL 语法（增删改查、连接、子查询）
  - MySQL / PostgreSQL 基本使用
  - 简单数据库设计（主键、外键、索引）

## Spring 框架核心（前置知识）
- Spring Core
  - 控制反转（IoC）
  - 依赖注入（DI）
  - Bean 生命周期 & 作用域
- Spring AOP
  - 切面编程概念
  - @Aspect、@Before、@After 等注解
- Spring JDBC / JdbcTemplate
- Spring 事务管理（@Transactional）

## Spring Boot 入门
- 快速上手
  - Spring Initializr 创建项目
  - 自动配置原理（@SpringBootApplication）
  - Starter 依赖机制
- 核心功能
  - 配置文件（application.yml / .properties）
  - 外部化配置（@ConfigurationProperties, @Value）
  - Profile 多环境配置
- Web 开发基础
  - RESTful API 设计
  - @RestController / @RequestMapping
  - 参数绑定（@PathVariable, @RequestParam, @RequestBody）
  - 统一异常处理（@ControllerAdvice + @ExceptionHandler）

## 数据访问层
- Spring Data JPA
  - Repository 接口
  - 方法命名查询
  - @Entity、@Table、@Id 等注解
- MyBatis / MyBatis-Plus（可选）
  - XML 映射 vs 注解方式
  - 分页插件、逻辑删除
- 事务管理
  - 声明式事务（@Transactional）
  - 传播行为 & 隔离级别

## 进阶开发技能
- 安全控制
  - Spring Security
    - 用户认证（JWT / Session）
    - 权限控制（@PreAuthorize）
    - OAuth2 / RBAC 模型
- 异步与定时任务
  - @Async 异步调用
  - @Scheduled 定时任务
- 消息队列集成
  - RabbitMQ / Kafka
  - Spring Boot 整合消息中间件
- 缓存机制
  - Redis 集成（Spring Data Redis）
  - @Cacheable / @CacheEvict
- 日志与监控
  - SLF4J + Logback
  - Actuator 健康检查、指标监控
  - Prometheus + Grafana（可选）

## 微服务架构（精通阶段）
- Spring Cloud 生态
  - 服务注册与发现（Nacos / Eureka）
  - 负载均衡（Ribbon / LoadBalancer）
  - 服务调用（OpenFeign）
  - 网关（Spring Cloud Gateway）
  - 配置中心（Nacos Config / Apollo）
  - 熔断限流（Sentinel / Hystrix）
- 分布式事务（Seata）
- 单元测试与集成测试
  - JUnit 5 + Mockito
  - @SpringBootTest
- 容器化部署
  - Docker 打包 Spring Boot 应用
  - Docker Compose 编排
  - Kubernetes 基础（可选）

## 项目实战与最佳实践
- 项目结构规范（分层架构：controller / service / dao / dto）
- API 文档（Swagger / SpringDoc OpenAPI）
- 代码质量
  - Lombok 简化代码
  - MapStruct 对象映射
  - 统一返回格式（Result<T>）
- 性能优化
  - 连接池（HikariCP）
  - SQL 优化 & 索引
  - JVM 调优基础
- DevOps 实践
  - Git 版本控制
  - CI/CD（GitHub Actions / Jenkins）
  - 日志收集（ELK）

## 持续学习方向
- 深入源码：Spring Boot 自动配置原理、启动流程
- 响应式编程：Spring WebFlux
- 新特性跟进：Spring Boot 3.x + Jakarta EE 9+
- 架构设计：DDD（领域驱动设计）、CQRS
